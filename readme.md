## Getting Started<a name="getting-started"></a>
To get started with the game, simply open the `index.html` file in your preferred web browser.

```bash
open index.html
```
After opening the index.html file in your preferred web browser, the next steps depend on what you want to do:

## Configure Player Names:

The game will prompt you to configure player names. Click the "Edit" button for Player 1 and Player 2 to set your desired names.
Enter the player names in the provided form and click "Confirm" to save the changes.
Start a New Game:

After configuring player names, click the "Start New Game" button to begin the game.
## Play the Game:

The game will display an empty TicTacToe board.
Players take turns placing their symbols (X or O) on the board.
The game announces the winner when a player achieves three in a row horizontally, vertically, or diagonally.
## Start a New Game Again:

To start a new game after the current game is finished, click the "Start New Game" button again.
